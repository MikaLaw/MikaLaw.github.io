Мой процесс работы c ними

В верстке мне нравится сам процесс и видимый результат своей работы. Люблю пробовать разные подходы к решению задачи и оптимально использовать свои знания, совмещаю адаптивный и отзывчивый подход. Для сеток и элементов использую grid и flexbox, пробовала табличную верстку в рассылках, библиотеку Bootstrap, но предпочитаю БЭМ нейминг. Стремлюсь к качеству, проверяю свою верстку на валидность и не забываю про доступность. Чтобы мои сайты выглядели хорошо на любом устройстве применяю префиксы, использую доступные в большинстве браузеров свойства, помню о различиях работы свойств на десктопах и мобильных. Для быстроты работы сайта оптимизирую картинки, использую webp, медиазапросы, собираю иконки в спрайты, использую ленивую загрузку, а также собираю все стили и скрипты в один соответствующий файл, знаю про критический CSS и приоритетность загрузки.

JavaScript-ом оживляю страницы, работаю с DOM, фишками ES6, помогающими мне в работе с объектами и массивами. Применяла на сайтах и jQuery с неплохими плагинами для интерактивности. Делала также проекты с динамически генерируемым контентом на чистом js, в которых опробовала паттерн MVC и классы. Чтобы писать код быстрее использую препроцессоры такие как pug и SCSS. Проекты собираю на gulp, или webpack. С репозиториями работаю через git, загружала сайты на GitLab и GitHub.

Более сложные проекты делаю на React, для типизации данных использую TypeScript. Успела застать использование классовых компонентов, сейчас же работаю с функциональными и хуками. Для управления состоянием использую redux и сопутствующие библиотеки, такие как redux-sagas, thunk, toolkit. В проектах личных кабинетов понравилось использовать библиотеку antd. Для стилизации использую импорты файлов в компоненты, знакома также с css-модулями и css-in-js.

Работаю с запросами по архитектуре REST API, для проверки запросов использовала Postman и Swagger, для отправки запросов - Axios, Fetch.

Из редакторов кода и IDE предпочитаю VS code, для форматирования использую ESLint, StylrLint, Prettier. С макетами работаю в Figma, сверяю по Pixel Perfect.
